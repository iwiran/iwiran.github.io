# DEMO

直接发布 Markdown 文件的静态网站 DEMO，只需在 _config.yml 文件中声明 `theme: jekyll-theme-cayman`，主题自动应用于存储库中的 Markdown 文件。

主页地址：https://iwiran.github.io/

更多示范地址：

 - Markdown https://iwiran.github.io/doc/1201
