GitHub Pages

在 GitHub Pages 上有两种方式呈现您的静态网站。

1、直接发布Markdown文档

这是一种最简单的方法，直接往 Github 存储库中添加 .md 文档，并在项目设置中选择一个美化主题，建立一个文档索引页面即可。GitHub Pages 可以直接将 .md 文件渲染成网页。

如: 我是怎么开发智能挠猫机的、新年快乐。

2、静态网站生成器

Jekyll（或者 Hugo / Hexo）通过标记语言和模板引擎转换生成网页，这需要您在本地搭建 Jekyll 运行环境，并将生成的站点文件 Push 到 Github 存储库。看起来麻烦很多，但可以实现更多的功能。
